# WordsVoyage

**WordsVoyage** is an app designed for users learning **any language** who want to collect words from **various sources** and use the app as a **personal vocabulary database**. Users can add their own words, organize them into categories, practice memorization, and use AI to generate new words.

## Features

- **Words** – Add your own words or generate them with AI based on your level of knowledge.
- **Categories** – Organize words by topics or languages.  
- **Exercises** – Translate word, multiple choice and translate sentence .

## Technologies

- **Frontend:** React, HTML, CSS, Vite
- **Backend:** Clojure, Leiningen

## How to Run

1. Start the database:
 - create the database named languagestudy in your PostgreSQL server;
 - initialize the database by running the SQL commands in database/init.sql;
   
2. Start the backend:
- lein run

3. Start the frontend:
- npm start
